# Archive

This directory preserves older experimental material that was removed from the main repo surface but not deleted.

## Structure

- `configs/`: old one-off model presets and tuning experiments
- `runs/`: raw historical run artifacts moved out of the active `runs/` area
- `debug/`: ad hoc analysis outputs such as parser replay reports

Archived files are kept for reference and auditability, but they are not part of the primary publishable pipeline.
